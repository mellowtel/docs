**Mellowtel ist die Open-Source-Monetarisierungsplattform für Entwickler und Kreative in der KI-Welt.** Füge das SDK in deine Browsererweiterung, Desktop-App, Website oder mobile App ein, und deine Nutzer können sich dafür entscheiden, einen Teil ihrer ungenutzten Internetbandbreite mit vertrauenswürdigen Partnern zu teilen. Du verdienst einen Anteil am Umsatz.

## So sieht die Integration aus

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

Das ist die gesamte Integration für eine Browsererweiterung. Andere Plattformen sind ähnlich. Sieh dir die Schnellstarts unten an.

## So funktioniert der Umsatz

Die Aufteilung ist **55% für Entwickler, 45% für Mellowtel**. Während wir wachsen, werden wir in der Lage sein, mehr Dienstleistungen anzubieten und den Entwickleranteil zu erhöhen. Zukünftig könnte es Funktionen geben, die es dir ermöglichen, einen Teil dieses Umsatzes direkt mit deinen Nutzern zu teilen; in der Zwischenzeit kannst du Nutzern, die sich angemeldet haben, exklusive Funktionen oder Inhalte bieten.
