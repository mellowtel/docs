**Mellowtel ist die Open-Source-Monetarisierungsplattform für Entwickler und Kreative in der KI-Welt.** Füge das SDK in deine Browser-Erweiterung, Desktop-App, Website oder mobile App ein und deine Nutzer können sich entscheiden, einen Teil ihrer ungenutzten Internetbandbreite mit vertrauenswürdigen Partnern zu teilen. Du erhältst einen Anteil am Umsatz.

## So sieht die Integration aus

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

Das ist die gesamte Integration für eine Browser-Erweiterung. Andere Plattformen sind ähnlich. Siehe die Schnellstarts unten.

## Wie der Umsatz funktioniert

Die Aufteilung ist **55% für Entwickler, 45% für Mellowtel**. Mit zunehmendem Wachstum werden wir in der Lage sein, mehr Dienste anzubieten und den Entwickleranteil zu erhöhen. In Zukunft könnte es Funktionen geben, die es dir ermöglichen, einen Teil dieses Umsatzes direkt mit deinen Nutzern zu teilen; bis dahin kannst du Nutzern, die sich angemeldet haben, exklusive Funktionen oder Inhalte anbieten.
