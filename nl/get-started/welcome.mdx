**Mellowtel is het open-source monetisatieplatform voor ontwikkelaars en makers in de AI-wereld.** Voeg de SDK toe aan je browserextensie, desktopapp, website of mobiele app en je gebruikers kunnen ervoor kiezen om een deel van hun ongebruikte internetbandbreedte te delen met vertrouwde partners. Jij verdient een deel van de inkomsten.

## Hoe integratie eruitziet

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

Dat is de hele integratie voor een browserextensie. Andere platforms zijn vergelijkbaar. Zie de quickstarts hieronder.

## Hoe de inkomsten werken

De verdeling is **55% naar ontwikkelaars, 45% naar Mellowtel**. Naarmate we groeien, zullen we meer diensten kunnen aanbieden en het aandeel voor ontwikkelaars vergroten. In de toekomst kunnen er functies zijn waarmee je een deel van deze inkomsten direct met je gebruikers kunt delen; in de tussentijd kun je gebruikers die hebben gekozen voor deelname belonen met exclusieve functies of inhoud.
