## NuGet CLI-installatie

Om Mellowtel in je Flutter-applicaties op Windows te integreren, heb je de NuGet CLI-tool nodig. Volg deze stappen:

1. Download `nuget.exe` van de [NuGet CLI downloadpagina](https://www.nuget.org/downloads)
2. Maak een map aan (bijv. `C:\Tools\NuGet`) en plaats `nuget.exe` daar
3. Voeg de map toe aan je Windows PATH:
   - Open Systeemeigenschappen > Geavanceerd > Omgevingsvariabelen
   - Onder Systeemvariabelen, bewerk Path
   - Voeg het mappad toe (bijv. `C:\Tools\NuGet`)
4. Verifieer de installatie door `nuget` in een nieuwe terminal uit te voeren

Voor gedetailleerde instructies, bezoek de [officiële NuGet CLI-documentatie](https://learn.microsoft.com/en-us/nuget/install-nuget-client-tools?tabs=windows#nugetexe-cli).

## Problemen?

Als je problemen tegenkomt, raadpleeg dan de [NuGet CLI-documentatie](https://learn.microsoft.com/en-us/nuget/install-nuget-client-tools?tabs=windows#nugetexe-cli) voor tips over probleemoplossing of neem contact op met ons ondersteuningsteam op [Discord](https://discord.com/invite/txAZp4MSDe).
