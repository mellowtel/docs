Als je extensie is gebouwd met behulp van het [WXT framework](https://wxt.dev/), kun je Mellowtel er eenvoudig in integreren.

Je kunt zien hoe je Mellowtel integreert in je met WXT gebouwde extensie door de gids op [Github](https://github.com/yacine-bens/mellowtel-wxt-samples) te volgen.

Als je vragen hebt of verdere hulp nodig hebt, sluit je dan aan bij onze [Discord-server](https://discord.com/invite/txAZp4MSDe) of neem contact op met de ontwikkelaar van deze repo via [yacine.web.ext@gmail.com](mailto:yacine.web.ext@gmail.com).
