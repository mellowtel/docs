Nadat je de instructies hebt gevolgd om Mellowtel in je extensie te importeren zoals beschreven in [Snelle Start](/browser-plugins/quickstart), ben je klaar om je extensie in te dienen bij de webwinkel.
Deze gids helpt je bij het indienen van je extensie bij de Chrome Web Store.

Geschatte tijd om te voltooien: 3-5 minuten.


### Webwinkel Rechtvaardiging
De webwinkel vereist dat je de toestemmingen die je aanvraagt rechtvaardigt. Hier is een sjabloon dat je kunt gebruiken (het legt uit hoe Mellowtel de toestemmingen gebruikt):

**DeclarativeNetRequest**:
```txt
De declarativeNetRequest toestemming is vereist om bepaalde headers (bijv. X-Frame headers) van sommige reacties op het sub_frame niveau te verwijderen. Dit maakt het mogelijk om websites in een iframe weer te geven zonder problemen met cross-origin resource sharing (CORS). Deze headers worden onmiddellijk hersteld nadat de reactie is verwerkt.
```

**Host Permissions**:
```txt
De host permissions op all_urls zijn vereist om declarativeNetRequest toe te staan om responsheaders op alle URL's te wijzigen. Dit is noodzakelijk om bepaalde headers (bijv. X-Frame headers) van sommige reacties op het sub_frame niveau te verwijderen. Dit maakt het mogelijk om websites in een iframe weer te geven zonder problemen met cross-origin resource sharing (CORS). Deze headers worden onmiddellijk hersteld nadat de reactie is verwerkt.
```

**Storage**:
```txt
De opslagtoestemming is vereist om de status specifiek voor de extensie te behouden en een lokale logica te hebben om communicatie tussen de verschillende delen van de extensie te vergemakkelijken.
```


### Dien je extensie in
Je kunt de instructies hier volgen, afhankelijk van of je een nieuwe extensie indient of een bestaande bijwerkt:
- [Dien een nieuwe extensie in](https://developer.chrome.com/docs/webstore/publish/)
- [Werk een bestaande extensie bij](https://developer.chrome.com/docs/webstore/update)
