**Mellowtel é a plataforma de monetização de código aberto para desenvolvedores e criadores no mundo da IA.** Insira o SDK na sua extensão de navegador, aplicativo de desktop, site ou aplicativo móvel e seus usuários podem optar por compartilhar uma fração de sua largura de banda de internet não utilizada com parceiros confiáveis. Você ganha uma parte da receita.

## Como é a integração

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

Essa é toda a integração para uma extensão de navegador. Outras plataformas são semelhantes. Veja os guias rápidos abaixo.

## Como funciona a receita

A divisão é **55% para desenvolvedores, 45% para Mellowtel**. À medida que crescemos, seremos capazes de oferecer mais serviços e aumentar a participação dos desenvolvedores. No futuro, pode haver recursos que permitam compartilhar parte dessa receita diretamente com seus usuários; enquanto isso, você pode recompensar os usuários que optaram por participar com recursos ou conteúdos exclusivos.
