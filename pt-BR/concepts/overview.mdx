O Mellowtel é construído sobre quatro princípios. Leia estes uma vez e você entenderá todos os comportamentos, padrões e escolhas de API no restante da documentação.

<CardGroup cols={2}>
  <Card title="Opt-in / Opt-out" icon="signature" href="/concepts/opt-in-out">
    Os usuários devem dar consentimento explícito antes de qualquer largura de banda ser compartilhada e podem revogá-lo a qualquer momento.
  </Card>
  <Card title="Privacidade" icon="lock" href="/concepts/privacy">
    As solicitações são executadas em um ambiente isolado, sem credenciais. Nenhum cookie, armazenamento ou dado pessoal jamais deixa o dispositivo.
  </Card>
  <Card title="Limitação de Taxa" icon="stopwatch" href="/concepts/rate-limiting">
    O SDK se regula automaticamente para que a experiência de navegação do usuário nunca seja afetada.
  </Card>
  <Card title="Código Aberto" icon="github" href="https://github.com/mellowtel-inc/">
    Cada linha está no GitHub. Se um documento não estiver claro, o código fonte é a verdade fundamental.
  </Card>
</CardGroup>
