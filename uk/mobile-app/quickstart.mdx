**Статус: Все ще в розробці**

Наша інтеграція з мобільним додатком наразі знаходиться в розробці. Якщо ти зацікавлений в інтеграції Mellowtel з твоїм мобільним додатком, будь ласка, зв'яжись з нами:

- Email: info@mellowtel.com
- Приєднуйся до нашої [спільноти в Discord](https://discord.com/invite/txAZp4MSDe)

Ми повідомимо тебе, як тільки SDK для мобільного додатка стане доступним.

## Заплановані функції

- Рідний SDK для iOS
- Рідний SDK для Android
- Підтримка React Native
- Кросплатформенна сумісність
- Аналітика в реальному часі
- Відстеження взаємодії з користувачами
