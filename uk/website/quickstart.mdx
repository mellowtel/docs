Mellowtel дозволяє вашим відвідувачам погоджуватися ділитися невикористаною пропускною здатністю в обмін на підтримку вас. Ви додаєте невеликий віджет на свій сайт, відвідувачі обирають встановити супутнє розширення, і ви заробляєте на основі запитів, які обробляє їхній браузер.

<Info>
  **Орієнтовний час:** 5-10 хвилин. **Вам знадобиться:** [обліковий запис Mellowtel](https://www.mellowtel.com/mellowtel-auth/) та доступ до HTML вашого сайту (або еквівалент у фреймворку).
</Info>

## Як це працює

<Steps>
  <Step title="Ви вбудовуєте віджет">
    Тег скрипта відображає кнопку, іконку або банер на вашому сайті.
  </Step>
  <Step title="Відвідувач натискає на нього">
    Його перенаправляють на встановлення супутнього розширення Mellowtel.
  </Step>
  <Step title="Ви отримуєте кредит">
    Ваш `invite_id` пов'язує цього відвідувача з вашим обліковим записом, тому запити, оброблені їхнім браузером, зараховуються вам.
  </Step>
</Steps>

## Перед початком

- Обліковий запис Mellowtel. [Зареєструйтеся тут](https://www.mellowtel.com/mellowtel-auth/), якщо у вас його ще немає.
- Ваш `invite_id`, який можна знайти на вашій [панелі управління](https://www.mellowtel.com/mellowtel-auth/) після створення посилання підтримки.
- Доступ до HTML вашого сайту або можливість додати тег `<script>` через ваш фреймворк або CMS.

## Встановлення віджета

Вставте це на ваш сайт, замінивши `INVITE_ID` значенням з вашої панелі управління. Рекомендується розміщувати його безпосередньо перед `</body>`, щоб він не блокував рендеринг сторінки.

<CodeGroup>

```html HTML
<script
  src="https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID"
  async>
</script>
```

```jsx Next.js (App Router)
import Script from 'next/script'

export default function Layout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Script
          src="https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID"
          strategy="afterInteractive"
        />
      </body>
    </html>
  )
}
```

```jsx React (Vite / CRA)
import { useEffect } from 'react'

export function MellowtelWidget() {
  useEffect(() => {
    const s = document.createElement('script')
    s.src = 'https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID'
    s.async = true
    document.body.appendChild(s)
    return () => { s.remove() }
  }, [])
  return null
}
```

```html WordPress
<!-- Додайте до footer.php вашої теми безпосередньо перед </body>, -->
<!-- або використовуйте плагін "Header & Footer Scripts" і вставте у поле для футера. -->
<script
  src="https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID"
  async>
</script>
```

```html Webflow / Shopify
<!-- Webflow: Project Settings > Custom Code > Footer Code -->
<!-- Shopify: Online Store > Themes > Edit Code > theme.liquid (перед </body>) -->
<script
  src="https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID"

```

</CodeGroup>

## Опції конфігурації

Керуйте зовнішнім виглядом віджета через параметри запиту в URL скрипта.

| Параметр    | Обов'язковий | Значення                                                           | Опис                                                  |
| ------------ | -------- | ------------------------------------------------------------------- | ---------------------------------------------------- |
| `invite_id`  | Так      | Ваш ID з панелі управління                                          | Зараховує заробіток на ваш рахунок.                 |
| `type`       | Так      | `button`, `icon`, `banner`                                          | Стиль віджета, що відображається на вашому сайті.                     |
| `position`   | Так      | `bottom-right`, `bottom-left`, `top-right`, `top-left`              | Де віджет розташований на сторінці.                   |

<Tip>
  Попередньо перегляньте кожну комбінацію в реальному часі перед публікацією: [відкрийте попередній перегляд віджета](https://cdn.mellow.tel/preview?position=bottom-right&type=button&invite_id=ijxf49s9h2f). Замініть параметри `type` та `position` в URL, щоб побачити кожен варіант.
</Tip>

## Вирішення проблем

<AccordionGroup>
  <Accordion title="Віджет не відображається">
    - Переконайтеся, що тег скрипта знаходиться всередині `<body>` і ваш `invite_id` заповнений.
    - Відкрийте DevTools і перевірте вкладку Мережа на наявність запиту до `cdn.mellow.tel/widget.js`. Якщо він заблокований, ймовірно, причина в блокувальнику реклами або фільтрі контенту.
    - Перевірте вашу Політику безпеки контенту. Можливо, вам потрібно дозволити `https://cdn.mellow.tel` у `script-src`.
  </Accordion>
  <Accordion title="Я бачу попередження 'відсутній invite_id'">
    Параметр `invite_id` порожній або все ще містить буквальний заповнювач `INVITE_ID`. Скопіюйте реальне значення з вашої панелі управління і вставте його в URL.
  </Accordion>
  <Accordion title="Це працює локально, але не в продакшені">
    Найчастіше правило CSP або CDN видаляє скрипт. Перевірте заголовки відповідей на вашому хості в продакшені і додайте в білий список `cdn.mellow.tel`.
  </Accordion>
  <Accordion title="Як видалити віджет?">
    Видаліть тег `<script>` (або компонент фреймворку, що його обгортає). Більше нічого не потрібно очищати.
  </Accordion>
</AccordionGroup>
