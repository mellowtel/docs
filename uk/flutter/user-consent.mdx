## Вступ

Щоб активувати Mellowtel, користувачі повинні дати свою явну згоду. Пакет надає спосіб підключення або відключення. Основна ідея полягає в тому, що оскільки це їхня інтернет-пропускна здатність, вони повинні вирішувати, чи хочуть вони її ділитися чи ні.

Користувачі можуть надати свою згоду через діалог налаштувань згоди.

### Конфігурація Діалогу Згоди

Діалог згоди є важливою частиною інтеграції Mellowtel. Він гарантує, що користувачі повністю поінформовані та надають явну згоду перед тим, як ділитися своєю інтернет-пропускною здатністю.

Ви можете визначити конфігурацію діалогу згоди при створенні екземпляра `Mellowtel`

```dart
import 'package:mellowtel/mellowtel.dart';

final Mellowtel mellowtel = Mellowtel(
    "your_configuration_key",
    dialogConfiguration: const ConsentDialogConfiguration(
        appName: 'Назва Вашого Додатку',
        incentive: 'Заробляйте винагороди, ділячись своїм інтернетом'
        appIcon: 'assets/logo.png', // Необов’язкова іконка додатку, що відображається в діалозі згоди. Допомагає створити довіру.
        acceptButtonText: "Так, почнемо!", // Необов’язковий текст для кнопки прийняття ("Так" за замовчуванням).
        declineButtonText: 'Не зараз', // Необов’язковий текст для кнопки відмови ("Ні" за замовчуванням).
        dialogTextOverride: 'Користувацьке повідомлення згоди', // Необов’язкове користувацьке або локалізоване повідомлення діалогу згоди.
    ),
);
```

#### Відкриття Діалогу Згоди

Діалог згоди за замовчуванням відкривається, коли викликається метод `start()`, якщо користувач ще не надав своїх уподобань.

```dart
await mellowtel.start(
    ... // інші налаштування
    onOptIn: () async {
        // Увімкніть винагороди, якщо користувач надав згоду. 
    }, 
    onOptOut: () async {
        // Вимкніть винагороди, якщо користувач відмовився/відкликав згоду.
    },
    showConsentDialog: true
);
```

Ви можете встановити `showConsentDialog` в false, якщо хочете керувати, де з’являється діалог. Ми рекомендуємо викликати метод `start` у двох різних місцях:

1. **При Запуску Додатку** з `showConsentDialog: false`, щоб почати роботу, якщо дозвіл вже надано.
2. **Після Першого "Ага" Моменту:** з `showConsentDialog: true`, щоб запросити згоду після того, як користувачі відчують цінність додатку.

Цей підхід забезпечує негайний початок роботи Mellowtel, отримуючи згоду користувача в оптимальний час.

### Діалог Налаштувань Згоди

Mellowtel забезпечує повний контроль і конфіденційність для ваших користувачів. Ваші користувачі можуть змінити свою згоду в будь-який час на сторінці налаштувань згоди. Ви можете надати її як опцію на сторінці налаштувань вашого додатку.

#### Показати Сторінку Налаштувань Згоди

Використовуйте метод `showConsentSettingsPage()`, щоб відкрити діалог налаштувань згоди.

```dart
await mellowtel.showConsentSettingsPage(
    context,
    onOptIn: () async {
        // Увімкніть винагороди, якщо користувач надав згоду. 
    }, 
    onOptOut: () async {
        // Вимкніть винагороди, якщо користувач відмовився/відкликав згоду.
    },
);
```

### Ручні Методи (Не Рекомендується)

На додаток до діалогу згоди та сторінки налаштувань, ви можете вручну керувати згодою користувача, використовуючи методи `optIn()` та `optOut()`.

#### Підключення

Вручну позначте користувача як підключеного.
```dart
await mellowtel.optIn();
```
#### Відключення

Вручну позначте користувача як відключеного.
```dart
await mellowtel.optOut();
```
#### Перевірка Статусу Згоди

Ви можете перевірити поточний статус згоди користувача, використовуючи метод `checkConsent()`.
```dart
bool? hasOptedIn = await mellowtel.checkConsent();
```
