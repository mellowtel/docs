Після імпорту Mellowtel у ваше розширення, як описано в [Швидкому старті](/browser-plugins/quickstart), виконайте наступні кроки, щоб подати ваше розширення до магазину Microsoft Edge Add-ons.

## Підготовка вашого розширення

1. **Розробка та тестування**: Переконайтеся, що у вас є робочий прототип вашого розширення, протестований у Microsoft Edge.

2. **Створення ZIP-файлу**: Упакуйте файли вашого розширення в архів ZIP, включаючи:
   - Маніфест розширення
   - Усі необхідні зображення та файли

3. **Перевірка маніфесту**: Переконайтеся, що наступні поля у вашому маніфесті є точними:
   - `Name`
   - `Description`

## Подання до Microsoft Edge Add-ons

1. **Створення облікового запису розробника**:
   - Якщо у вас його немає, створіть обліковий запис розробника на [Partner Center](https://partner.microsoft.com/dashboard/microsoftedge/public/login?ref=addons).
   - Для детальних інструкцій щодо створення облікового запису розробника, дивіться [Посібник з реєстрації розробника](https://docs.microsoft.com/microsoft-edge/extensions-chromium/publish/create-dev-account).

2. **Початок нового подання**:
   - Перейдіть на панель розробника та виберіть "Створити нове розширення".

3. **Завантаження вашого пакету**:
   - На сторінці Пакети завантажте ваш ZIP-файл розширення.
   - Дочекайтеся перевірки та вирішіть будь-які проблеми, якщо це необхідно.

   ![Екран завантаження пакету](/images/edge/packages.png)
   
   Екран завантаження дозволяє перетягнути ваш ZIP-файл або вибрати його. Переконайтеся, що ваш пакет містить:
   1. Файл маніфесту з такими деталями, як ім'я, версія, короткий опис, дозволи та мова за замовчуванням.
   2. Зображення та інші файли, які потребує ваше розширення.

4. **Надання деталей доступності**:
   - Встановіть видимість (Публічна або Прихована).
   - Виберіть ринки для розповсюдження.

   ![Налаштування доступності](/images/edge/availability.png)
   
   У розділі Доступність:
   - Виберіть між Публічною (будь-хто може знайти та використовувати розширення) або Прихованою (лише ті, хто має URL списку магазину, можуть використовувати його).
   - Виберіть ринки, на яких ваше розширення буде доступним. За замовчуванням встановлено всі поточні та майбутні ринки.

5. **Встановлення властивостей розширення**:
   - Виберіть категорію.
   - Вкажіть вимоги до політики конфіденційності.
   - Додайте URL вебсайту та контактні дані підтримки.
   - Вкажіть, чи є вміст для дорослих.

   ![Розділ властивостей](/images/edge/properties.png)
   
   У розділі Властивості:
   - Виберіть категорію, яка найкраще описує ваше розширення.
   - Вкажіть, чи ваше розширення отримує доступ, збирає або передає особисту інформацію.
   - Надайте URL політики конфіденційності, якщо це потрібно.
   - Додайте URL вашого вебсайту та контактні дані підтримки.

6. **Додавання деталей списку магазину**:
   - Надайте опис (до 10,000 символів).
   - Завантажте логотип магазину розширень (рекомендується 300x300 пікселів).
   - Додайте необов'язкові промоційні плитки та скріншоти.
   - Включіть пошукові терміни для покращення видимості.

7. **Завершення подання**:
   - Додайте будь-які примітки для сертифікації.
   - Натисніть "Опублікувати", щоб подати ваше розширення.

## Після подання

- Microsoft перевірить ваше розширення, що може зайняти до семи робочих днів.
- Ви будете повідомлені, коли перевірка буде завершена.
- Якщо будуть виявлені проблеми, вирішіть їх і подайте знову.

## Поради для успішного подання

- Дотримуйтесь [Політик розробника для Microsoft Edge Add-ons](https://docs.microsoft.com/microsoft-edge/extensions-chromium/store-policies/developer-policies).
- Надавайте чіткі, стисливі описи та якісні зображення.
- Переконайтеся, що ваша політика конфіденційності є всеосяжною, якщо ви збираєте будь-які дані користувачів.
- Швидко відповідайте на будь-які коментарі чи запитання від рецензентів.

Пам'ятайте, інтеграція Mellowtel не повинна впливати на процес подання, але переконайтеся, що вона відповідає політикам Microsoft щодо збору даних та конфіденційності користувачів.

Для детальнішої інформації відвідайте [Документацію розробника Microsoft Edge Add-ons](https://docs.microsoft.com/microsoft-edge/extensions-chromium/publish/publish-extension).
