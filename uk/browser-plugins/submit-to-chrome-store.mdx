Після того, як ти завершиш дотримання інструкцій для імпорту Mellowtel у своє розширення, які були описані в [Швидкому старті](/browser-plugins/quickstart), ти готовий надіслати своє розширення до веб-магазину.
Цей посібник допоможе тобі надіслати своє розширення до Chrome Web Store.

Орієнтовний час завершення: 3-5 хвилин.


### Обґрунтування для веб-магазину
Веб-магазин вимагає обґрунтування дозволів, які ти запитуєш. Ось шаблон, який ти можеш використовувати (він пояснює, як Mellowtel використовує дозволи):

**DeclarativeNetRequest**:
```txt
Дозвіл declarativeNetRequest потрібен для видалення певних заголовків (наприклад, X-Frame заголовків) з деяких відповідей на рівні sub_frame. Це дозволяє відображати веб-сайти в iframe без проблем, пов'язаних з обміном ресурсами між різними джерелами (CORS). Ці заголовки негайно відновлюються після обробки відповіді.
```

**Дозволи хоста**:
```txt
Дозволи хоста на all_urls потрібні, щоб дозволити declarativeNetRequest змінювати заголовки відповідей на всіх URL-адресах. Це необхідно для видалення певних заголовків (наприклад, X-Frame заголовків) з деяких відповідей на рівні sub_frame. Це дозволяє відображати веб-сайти в iframe без проблем, пов'язаних з обміном ресурсами між різними джерелами (CORS). Ці заголовки негайно відновлюються після обробки відповіді.
```

**Зберігання**:
```txt
Дозвіл на зберігання потрібен для збереження стану, специфічного для розширення, і для наявності локальної логіки, що полегшує комунікацію між різними частинами розширення.
```


### Надішли своє розширення
Ти можеш дотримуватися інструкцій тут, залежно від того, чи надсилаєш нове розширення, чи оновлюєш існуюче:
- [Надіслати нове розширення](https://developer.chrome.com/docs/webstore/publish/)
- [Оновити існуюче розширення](https://developer.chrome.com/docs/webstore/update)
