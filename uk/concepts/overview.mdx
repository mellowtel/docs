Mellowtel побудований на чотирьох принципах. Прочитай їх один раз, і ти зрозумієш кожну поведінку, значення за замовчуванням та вибір API в решті документації.

<CardGroup cols={2}>
  <Card title="Опт-ін / Опт-аут" icon="signature" href="/concepts/opt-in-out">
    Користувачі повинні дати явну згоду, перш ніж будь-яка пропускна здатність буде поділена, і можуть відкликати її в будь-який час.
  </Card>
  <Card title="Конфіденційність" icon="lock" href="/concepts/privacy">
    Запити виконуються в ізольованому середовищі без облікових даних. Жодні файли cookie, сховище чи особисті дані ніколи не залишають пристрій.
  </Card>
  <Card title="Обмеження швидкості" icon="stopwatch" href="/concepts/rate-limiting">
    SDK автоматично обмежує себе, щоб користувацький досвід перегляду ніколи не був порушений.
  </Card>
  <Card title="Відкритий код" icon="github" href="https://github.com/mellowtel-inc/">
    Кожен рядок є на GitHub. Якщо документ незрозумілий, джерело є основою істини.
  </Card>
</CardGroup>
