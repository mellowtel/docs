Mellowtel базується на чотирьох принципах. Прочитай їх один раз, і ти зрозумієш кожну поведінку, налаштування за замовчуванням і вибір API в решті документації.

<CardGroup cols={2}>
  <Card title="Згода / Відмова" icon="signature" href="/concepts/opt-in-out">
    Користувачі повинні дати явну згоду перед тим, як будь-яка пропускна здатність буде поділена, і можуть відкликати її в будь-який час.
  </Card>
  <Card title="Приватність" icon="lock" href="/concepts/privacy">
    Запити виконуються в ізольованому середовищі без облікових даних. Жодні файли cookie, сховище чи особисті дані ніколи не залишають пристрій.
  </Card>
  <Card title="Обмеження швидкості" icon="stopwatch" href="/concepts/rate-limiting">
    SDK автоматично обмежує себе, щоб досвід перегляду користувача ніколи не постраждав.
  </Card>
  <Card title="Відкритий код" icon="github" href="https://github.com/mellowtel-inc/">
    Кожен рядок коду є на GitHub. Якщо документ неясний, вихідний код є основою істини.
  </Card>
</CardGroup>
