Esta guía te llevará a través de la integración de Mellowtel en tu extensión de Firefox, compatible con Manifest V2 y V3.

## Instalación

Primero, instala el paquete npm Mellowtel Lite (`mellowtel-lite`). Ten en cuenta que para Firefox solo puedes importar el paquete `mellowtel-lite` para cumplir con las políticas del navegador respecto a los datos auxiliares del usuario:

```bash
npm install mellowtel-lite
```

## Configuración del Manifest

El archivo `manifest.json` necesita configuraciones específicas para que Mellowtel funcione correctamente. Aquí están las partes requeridas para ambos Manifest V2 y V3:

### Manifest V2

```json
{
  "manifest_version": 2,
  "permissions": [
    "storage",
    "<all_urls>",
    "declarativeNetRequestWithHostAccess",
  ],
  "background": {
    "scripts": ["background.js"],
    "persistent": true
  },
  "content_scripts": [
    {
      "matches": ["<all_urls>"],
      "js": ["content_start_mellowtel.js"],
      "run_at": "document_start",
      "all_frames": true
    }
  ],
  "browser_action": {
    "default_popup": "popup.html"
  }
}
```

### Manifest V3

```json
{
  "manifest_version": 3,
  "permissions": [
    "storage",
    "declarativeNetRequestWithHostAccess"
  ],
  "host_permissions": [
    "<all_urls>"
  ],
  "background": {
    "scripts": ["background.js"],
    "type": "module"
  },
  "content_scripts": [
    {
      "matches": ["<all_urls>"],
      "js": ["content_start_mellowtel.js"],
      "run_at": "document_start",
      "all_frames": true
    }
  ],
  "action": {
    "default_popup": "popup.html"
  }
}
```

## Configuración del Script de Fondo

Crea un archivo llamado `background.js` con el siguiente contenido:

```javascript
import Mellowtel from "mellowtel-lite";

let mellowtel;

(async () => {
    mellowtel = new Mellowtel("<your_configuration_key>");
    await mellowtel.initBackground();
})();

browser.runtime.onInstalled.addListener(async function(details) {
    console.log("Extensión instalada o actualizada");
    await mellowtel.generateAndOpenOptInLink();
});
```

Reemplaza `<your_configuration_key>` con tu clave de configuración de Mellowtel real.

## Configuración del Script de Contenido

Crea un archivo llamado `content_start_mellowtel.js` con el siguiente contenido:

```javascript
import Mellowtel from "mellowtel-lite";

(async () => {
    const mellowtel = new Mellowtel("<your_configuration_key>");
    await mellowtel.initContentScript();
})();
```

Nuevamente, reemplaza `<your_configuration_key>` con tu clave de configuración de Mellowtel real.

## Popup para Configuraciones de Mellowtel

Para permitir que los usuarios accedan fácilmente a las configuraciones de Mellowtel, debes crear un popup. Aquí te explicamos cómo configurarlo:

1. Crea un archivo `popup.html`:

```html
<!DOCTYPE html>
<html>
<head>
  <title>Configuraciones de Mellowtel</title>
</head>
<body>
  <h1>Configuraciones de Mellowtel</h1>
  <button id="openSettings">Abrir Configuraciones de Mellowtel</button>
  <script src="popup.js"></script>
</body>
</html>
```

2. Crea un archivo `popup.js`:

```javascript
import Mellowtel from "mellowtel-lite";

const mellowtel = new Mellowtel("<your_configuration_key>");

async function openSettings() {
    try {
        const settingsLink = await mellowtel.generateSettingsLink();
        browser.tabs.create({ url: settingsLink });
    } catch (error) {
        console.error("Error al generar el enlace de configuraciones:", error);
    }
}

document.getElementById('openSettings').addEventListener('click', openSettings);
```

Recuerda reemplazar `<your_configuration_key>` con tu clave de configuración de Mellowtel real. Si aún no tienes una clave de configuración, puedes obtener una registrándote en [Mellowtel](https://www.mellowtel.com/mellowtel-auth/).
