**Mellowtel es la plataforma de monetización de código abierto para desarrolladores y creadores en el mundo de la IA.** Inserta el SDK en tu extensión de navegador, aplicación de escritorio, sitio web o aplicación móvil y tus usuarios pueden optar por compartir una fracción de su ancho de banda de internet no utilizado con socios de confianza. Tú ganas una parte de los ingresos.

## Cómo se ve la integración

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

Esa es toda la integración para una extensión de navegador. Otras plataformas son similares. Consulta los inicios rápidos a continuación.

## Cómo funcionan los ingresos

La división es **55% para los desarrolladores, 45% para Mellowtel**. A medida que escalamos, podremos ofrecer más servicios y aumentar la participación de los desarrolladores. En el futuro, puede haber funciones que te permitan compartir parte de estos ingresos directamente con tus usuarios; mientras tanto, puedes recompensar a los usuarios que opten por participar con funciones o contenido exclusivo.
