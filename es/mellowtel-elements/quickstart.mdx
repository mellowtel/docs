Mellowtel Elements proporciona componentes UI listos para usar que se pueden integrar tanto en extensiones de navegador como en sitios web. Estos componentes manejan el consentimiento del usuario y la gestión de configuraciones con un esfuerzo mínimo de integración.

## Características Clave

- **Integración Fácil** - Funciona tanto en extensiones como en sitios web
- **Estilo Personalizable** - Colores, dimensiones y apariencia
- **Validación** - Validación de entrada y manejo de errores integrados
- **Comunicación Segura** - Orientación específica de la extensión a través de ID y clave de configuración
