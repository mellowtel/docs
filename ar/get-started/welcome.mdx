**Mellowtel هو منصة مفتوحة المصدر لتحقيق الدخل للمطورين والمبدعين في عالم الذكاء الاصطناعي.** ضيف الـ SDK في إضافة المتصفح، تطبيق سطح المكتب، الموقع الإلكتروني، أو تطبيق الهاتف النقال، ويمكن للمستخدمين ديالك يختارو يشاركو جزء صغير من عرض النطاق الترددي غير المستخدم ديالهم مع شركاء موثوقين. انت تربح حصة من الإيرادات.

## كيفاش كتشوف التكامل

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

هذا هو التكامل الكامل لإضافة المتصفح. المنصات الأخرى مشابهة. شوف البدايات السريعة تحت.

## كيفاش كيتخدم الإيرادات

التقسيم هو **55% للمطورين، 45% لـ Mellowtel**. مع التوسع غادي نقدر نعرضو خدمات أكثر ونزيدو حصة المطورين. في المستقبل يمكن يكونو ميزات اللي تخليك تشارك جزء من هذ الإيرادات مباشرة مع المستخدمين ديالك؛ في الوقت الحالي تقدر تكافئ المستخدمين اللي اختارو المشاركة بميزات أو محتوى حصري.
