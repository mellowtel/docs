**Mellowtel هي منصة مفتوحة المصدر للربح للمطورين والمبدعين في عالم الذكاء الاصطناعي.** ضيف الـ SDK في الإضافة تاع المتصفح، تطبيق سطح المكتب، الموقع الإلكتروني، أو التطبيق المحمول تاعك والمستخدمين تاعك يقدرو يختارو يشاركو جزء صغير من النطاق الترددي للإنترنت اللي ما يستعملوش مع شركاء موثوقين. تقدر تربح حصة من الإيرادات.

## كيفاش تكون التكامل

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

هادي هي التكامل كامل لإضافة المتصفح. المنصات الأخرى مشابهة. شوف البدايات السريعة اللي تحت.

## قبل ما تطلق، اقرا المفاهيم

هاد الصفحات الأربعة تشرح المبادئ اللي لازم أي تكامل مع Mellowtel يحترمها: الموافقة، الخصوصية، وتحديد السرعة. اقراهم مرة وحدة وباقي الوثائق راح تكون مفهومة.

## كيفاش تخدم الإيرادات

التقسيم هو **55% للمطورين، 45% لـ Mellowtel**. كي نكبر نقدر نقدمو خدمات أكثر ونزيدو حصة المطورين. في المستقبل ممكن تكون ميزات تخليك تشارك جزء من هاد الإيرادات مباشرة مع المستخدمين تاعك؛ في الوقت الحالي تقدر تكافئ المستخدمين اللي اختارو الاشتراك بميزات حصرية أو محتوى.
