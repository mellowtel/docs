## مقدمة

باش تفعل Mellowtel، لازم المستخدمين يعطيو الموافقة ديالهم بشكل صريح. الحزمة توفر طريقة باش يقدرو يدخلو ولا يخرجو. الفكرة الأساسية هي بما أنو هاد الشيء يتعلق بالنطاق الترددي ديال الإنترنت ديالهم، لازم هما يقررو واش يبغيو يشاركوه ولا لا.

المستخدمين يقدرو يعطيو الموافقة ديالهم عبر حوار إعدادات الموافقة.

### إعدادات حوار الموافقة

حوار الموافقة هو جزء مهم من دمج Mellowtel. يضمن أن المستخدمين يكونو على علم تام ويعطيو الموافقة ديالهم بشكل صريح قبل ما يشاركو النطاق الترددي ديال الإنترنت ديالهم.

تقدر تحدد إعدادات حوار الموافقة ملي تنشئ نسخة من `Mellowtel`

```dart
import 'package:mellowtel/mellowtel.dart';

final Mellowtel mellowtel = Mellowtel(
    "your_configuration_key",
    dialogConfiguration: const ConsentDialogConfiguration(
        appName: 'اسم التطبيق ديالك',
        incentive: 'ربح مكافآت بمشاركة الإنترنت ديالك'
        appIcon: 'assets/logo.png', // أيقونة التطبيق الاختيارية اللي تتعرض في حوار الموافقة. تساعد في بناء الثقة.
        acceptButtonText: "نعم، نبدأ!", // نص اختياري لزر القبول ("نعم" بشكل افتراضي).
        declineButtonText: 'مش دابا', // نص اختياري لزر الرفض ("لا" بشكل افتراضي).
        dialogTextOverride: 'رسالة موافقة مخصصة', // رسالة حوار الموافقة المخصصة أو المحلية.
    ),
);
```

#### فتح حوار الموافقة

حوار الموافقة يفتح بشكل افتراضي ملي تتنادى على الطريقة `start()` إلا إذا المستخدم ما عطاش التفضيل ديالو بعد.

```dart
await mellowtel.start(
    ... // إعدادات أخرى
    onOptIn: () async {
        // تفعيل المكافآت إذا المستخدم عطى الموافقة ديالو.
    }, 
    onOptOut: () async {
        // تعطيل المكافآت إذا المستخدم رفض/سحب الموافقة ديالو.
    },
    showConsentDialog: true
);
```

تقدر تحدد `showConsentDialog` لـ false إذا كنت تبغي تدير وين يظهر الحوار. نوصيك باش تنادى على الطريقة `start` في جوج مواقع مختلفة:

1. **عند إطلاق التطبيق** مع `showConsentDialog: false` باش تبدأ الخدمة إذا كان الإذن معطى بالفعل.
2. **بعد أول لحظة "آها":** مع `showConsentDialog: true` باش تطلب الموافقة بعد ما المستخدمين يجربو قيمة التطبيق.

هاد النهج يضمن أن Mellowtel تبدأ الخدمة مباشرة بينما تحصل على موافقة المستخدم في الوقت المناسب.

### حوار إعدادات الموافقة

Mellowtel تضمن التحكم الكامل والخصوصية للمستخدمين ديالك. المستخدمين ديالك يقدرو يغيرو الموافقة ديالهم في أي وقت من صفحة إعدادات الموافقة. تقدر توفرها كخيار داخل صفحة الإعدادات ديال التطبيق ديالك.

#### عرض صفحة إعدادات الموافقة

استعمل الطريقة `showConsentSettingsPage()` باش تفتح حوار إعدادات الموافقة.

```dart
await mellowtel.showConsentSettingsPage(
    context,
    onOptIn: () async {
        // تفعيل المكافآت إذا المستخدم عطى الموافقة ديالو.
    }, 
    onOptOut: () async {
        // تعطيل المكافآت إذا المستخدم رفض/سحب الموافقة ديالو.
    },
);
```

### الطرق اليدوية (غير موصى بها)

بالإضافة إلى حوار الموافقة وصفحة الإعدادات، تقدر تدير الموافقة ديال المستخدم يدويا باستعمال الطرق `optIn()` و `optOut()`.

#### Opt-In

تحديد المستخدم يدويا كمنضم.
```dart
await mellowtel.optIn();
```
#### Opt-Out

تحديد المستخدم يدويا كمنسحب.
```dart
await mellowtel.optOut();
```
#### التحقق من حالة الموافقة

تقدر تتحقق من حالة الموافقة الحالية ديال المستخدم باستعمال الطريقة `checkConsent()`.
```dart
bool? hasOptedIn = await mellowtel.checkConsent();
```
