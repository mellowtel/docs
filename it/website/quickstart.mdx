Mellowtel permette ai tuoi visitatori di scegliere di condividere la larghezza di banda inutilizzata in cambio di supportarti. Aggiungi un piccolo widget al tuo sito, i visitatori scelgono di installare l'estensione complementare e guadagni in base alle richieste gestite dal loro browser.

<Info>
  **Tempo stimato:** 5 a 10 minuti. **Avrai bisogno di:** un [account Mellowtel](https://www.mellowtel.com/mellowtel-auth/) e accesso all'HTML del tuo sito (o un equivalente del framework).
</Info>

## Come funziona

<Steps>
  <Step title="Incorpori il widget">
    Un tag script visualizza un pulsante, un'icona o un banner sul tuo sito.
  </Step>
  <Step title="Un visitatore ci clicca sopra">
    Viene portato a installare l'estensione complementare di Mellowtel.
  </Step>
  <Step title="Ricevi credito">
    Il tuo `invite_id` collega quel visitatore al tuo account, quindi le richieste gestite dal loro browser sono attribuite a te.
  </Step>
</Steps>

## Prima di iniziare

- Un account Mellowtel. [Registrati qui](https://www.mellowtel.com/mellowtel-auth/) se non ne hai uno.
- Il tuo `invite_id`, trovato sulla tua [dashboard](https://www.mellowtel.com/mellowtel-auth/) dopo aver creato un link di supporto.
- Accesso all'HTML del tuo sito, o la possibilità di aggiungere un tag `<script>` tramite il tuo framework o CMS.

## Installa il widget

Incolla questo nel tuo sito, sostituendo `INVITE_ID` con il valore dalla tua dashboard. Si consiglia di posizionarlo appena prima di `</body>` per non bloccare il rendering della pagina.

<CodeGroup>

```html HTML
<script
  src="https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID"
  async>
</script>
```

```jsx Next.js (App Router)
import Script from 'next/script'

export default function Layout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Script
          src="https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID"
          strategy="afterInteractive"
        />
      </body>
    </html>
  )
}
```

```jsx React (Vite / CRA)
import { useEffect } from 'react'

export function MellowtelWidget() {
  useEffect(() => {
    const s = document.createElement('script')
    s.src = 'https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID'
    s.async = true
    document.body.appendChild(s)
    return () => { s.remove() }
  }, [])
  return null
}
```

```html WordPress
<!-- Aggiungi al footer.php del tuo tema appena prima di </body>, -->
<!-- o usa un plugin "Header & Footer Scripts" e incolla nel campo del footer. -->
<script
  src="https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID"
  async>
</script>
```

```html Webflow / Shopify
<!-- Webflow: Impostazioni Progetto > Codice Personalizzato > Codice Footer -->
<!-- Shopify: Negozio Online > Temi > Modifica Codice > theme.liquid (prima di </body>) -->
<script
  src="https://cdn.mellow.tel/widget.js?type=button&position=bottom-right&invite_id=INVITE_ID"
  async>
</script>
```

</CodeGroup>

## Opzioni di configurazione

Controlla l'aspetto del widget tramite i parametri di query nell'URL dello script.

| Parametro    | Obbligatorio | Valori                                                              | Descrizione                                          |
| ------------ | ------------ | ------------------------------------------------------------------- | ---------------------------------------------------- |
| `invite_id`  | Sì           | Il tuo ID dalla dashboard                                           | Attribuisce i guadagni al tuo account.               |
| `type`       | Sì           | `button`, `icon`, `banner`                                          | Stile del widget mostrato sul tuo sito.              |
| `position`   | Sì           | `bottom-right`, `bottom-left`, `top-right`, `top-left`              | Dove si posiziona il widget sulla pagina.            |

<Tip>
  Anteprima di ogni combinazione dal vivo prima di pubblicare: [apri l'anteprima del widget](https://cdn.mellow.tel/preview?position=bottom-right&type=button&invite_id=ijxf49s9h2f). Cambia i parametri `type` e `position` nell'URL per vedere ogni variante.
</Tip>

## Verifica che funzioni

<Steps>
  <Step title="Carica il tuo sito">
    Apri la pagina in una nuova scheda del browser (o in una finestra in incognito per bypassare la cache).
  </Step>
  <Step title="Conferma che il widget si visualizzi">
    Dovresti vedere il pulsante, l'icona o il banner nella posizione scelta. Se manca, controlla la console del browser per richieste bloccate.
  </Step>
  <Step title="Clicca attraverso il flusso">
    Cliccando sul widget dovresti essere portato alla pagina di installazione dell'estensione con il tuo `invite_id` allegato.
  </Step>
  <Step title="Controlla la tua dashboard">
    Una volta che un visitatore installa e opta per partecipare, la sua attività appare sulla tua dashboard Mellowtel.
  </Step>
</Steps>

## Risoluzione dei problemi

<AccordionGroup>
  <Accordion title="Il widget non appare">
    - Conferma che il tag script sia all'interno di `<body>` e che il tuo `invite_id` sia compilato.
    - Apri DevTools e controlla la scheda Network per una richiesta a `cdn.mellow.tel/widget.js`. Se è bloccata, probabilmente è causata da un blocco pubblicità o un filtro contenuti.
    - Controlla la tua Content Security Policy. Potresti dover permettere `https://cdn.mellow.tel` sotto `script-src`.
  </Accordion>
  <Accordion title="Vedo un avviso 'missing invite_id'">
    Il parametro `invite_id` è vuoto o contiene ancora il segnaposto letterale `INVITE_ID`. Copia il valore reale dalla tua dashboard e incollalo nell'URL.
  </Accordion>
  <Accordion title="Funziona in locale ma non in produzione">
    Molto spesso una regola CSP o CDN sta rimuovendo lo script. Controlla gli header di risposta sul tuo host di produzione e inserisci `cdn.mellow.tel` nella lista consentita.
  </Accordion>
  <Accordion title="Come rimuovo il widget?">
    Elimina il tag `<script>` (o il componente del framework che lo avvolge). Non c'è nient'altro da pulire.
  </Accordion>
</AccordionGroup>
