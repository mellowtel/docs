**Mellowtel è la piattaforma di monetizzazione open-source per sviluppatori e creatori nel mondo dell'IA.** Inserisci l'SDK nella tua estensione del browser, app desktop, sito web o app mobile e i tuoi utenti possono scegliere di condividere una frazione della loro larghezza di banda internet inutilizzata con partner fidati. Tu guadagni una parte del ricavo.

## Come appare l'integrazione

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

Questa è l'intera integrazione per un'estensione del browser. Altre piattaforme sono simili. Consulta le guide rapide qui sotto.

## Come funziona il ricavo

La divisione è **55% agli sviluppatori, 45% a Mellowtel**. Man mano che cresciamo saremo in grado di offrire più servizi e aumentare la quota per gli sviluppatori. In futuro potrebbero esserci funzionalità che ti permetteranno di condividere parte di questo ricavo direttamente con i tuoi utenti; nel frattempo puoi premiare gli utenti che hanno optato per l'integrazione con funzionalità o contenuti esclusivi.
