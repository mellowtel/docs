Mellowtel lets your users opt in to share a small fraction of their unused internet bandwidth with trusted partners (startups, researchers, and non-profits) that retrieve publicly available data from the web. You get paid for the requests served, users get whatever reward or premium feature you choose to offer them in return, and no personal data is collected, shared, or sold.

## Prerequisites

Before you start, make sure you have:

- A Mellowtel account and a **configuration key**. Generate yours from the [Mellowtel dashboard](https://www.mellowtel.com/mellowtel-auth/).
- A supported desktop framework (see below). Each framework guide includes a one-click reveal for the package registry install token, so you do not need to request one ahead of time.

<Warning>
  **User consent is mandatory.** Both SDKs enforce opt-in at the API level. Your app must display a clear consent dialog and must let users manage their opt-in state at any time.
</Warning>
