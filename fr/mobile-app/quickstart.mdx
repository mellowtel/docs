**Statut : Toujours en développement**

Notre intégration d'application mobile est actuellement en cours de développement. Si tu es intéressé par l'intégration de Mellowtel avec ton application mobile, merci de nous contacter :

- Email : info@mellowtel.com
- Rejoins notre [communauté Discord](https://discord.com/invite/txAZp4MSDe)

Nous te tiendrons informé dès que le SDK pour application mobile sera disponible.

- SDK iOS natif
- SDK Android natif
- Support React Native
- Compatibilité multiplateforme
- Analyse en temps réel
- Suivi de l'engagement utilisateur
