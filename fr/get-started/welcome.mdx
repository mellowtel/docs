**Mellowtel est la plateforme de monétisation open-source pour les développeurs et créateurs dans le monde de l'IA.** Intègre le SDK dans ton extension de navigateur, application de bureau, site web ou application mobile et tes utilisateurs peuvent choisir de partager une fraction de leur bande passante internet inutilisée avec des partenaires de confiance. Tu gagnes une part des revenus.

## À quoi ressemble l'intégration

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

C'est toute l'intégration pour une extension de navigateur. Les autres plateformes sont similaires. Consulte les démarrages rapides ci-dessous.

## Comment fonctionnent les revenus

La répartition est de **55% pour les développeurs, 45% pour Mellowtel**. À mesure que nous nous développons, nous pourrons offrir plus de services et augmenter la part des développeurs. À terme, il pourrait y avoir des fonctionnalités qui te permettront de partager une partie de ces revenus directement avec tes utilisateurs ; en attendant, tu peux récompenser les utilisateurs ayant opté pour le partage avec des fonctionnalités ou du contenu exclusifs.
