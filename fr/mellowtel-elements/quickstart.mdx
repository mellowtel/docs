Mellowtel Elements fournit des composants UI prêts à l'emploi qui peuvent être intégrés à la fois dans les extensions de navigateur et les sites web. Ces composants gèrent le consentement des utilisateurs et la gestion des paramètres avec un effort d'intégration minimal.

## Caractéristiques principales

- **Intégration facile** - Fonctionne dans les extensions et les sites web
- **Style personnalisable** - Couleurs, dimensions et apparence
- **Validation** - Validation des entrées et gestion des erreurs intégrées
- **Communication sécurisée** - Ciblage spécifique aux extensions via ID et clé de configuration
