Mellowtel 允许你的用户选择分享他们未使用的少量互联网带宽给可信赖的合作伙伴（初创公司、研究人员和非营利组织），这些合作伙伴从网络上检索公开可用的数据。你可以通过已处理的请求获得报酬，用户可以获得你选择提供给他们的奖励或高级功能，同时不会收集、分享或出售任何个人数据。

## 先决条件

在开始之前，请确保你已具备：

- 一个 Mellowtel 账户和一个**配置密钥**。从 [Mellowtel 仪表盘](https://www.mellowtel.com/mellowtel-auth/)生成你的密钥。
- 一个支持的桌面框架（见下文）。每个框架指南都包括一个一键显示的包注册表安装令牌，因此你无需提前请求。

<Warning>
  **用户同意是强制性的。** 两个 SDK 在 API 级别强制要求用户选择加入。你的应用必须显示一个明确的同意对话框，并且必须让用户随时管理他们的选择加入状态。
</Warning>

## 选择你的框架

选择你正在使用的框架并按照说明进行操作。

  <Card title="Windows (.NET)" icon="windows" href="/desktop-app/windows">
