在按照[快速开始](/browser-plugin/quickstart)指南将 Mellowtel 导入到你的扩展后，按照以下步骤将你的扩展提交到 Firefox 附加组件。

### 步骤 1：上传你的附加组件

1. 前往 [附加组件开发者中心](https://addons.mozilla.org/developers/)。
2. 点击“提交新附加组件”。
3. 你将看到这个页面：

![提交新附加组件页面](/images/firefox/submit_new.png)

4. 点击“选择文件...”并选择你的附加组件文件（.zip、.xpi、.crx、.jar 或 .xml）。
5. 点击“继续”并等待验证完成。

### 步骤 2：提交源代码

验证后，你会被询问是否需要提交源代码：

![源代码提交页面](/images/firefox/source_code.png)

1. 在“你需要提交源代码吗？”的问题中选择“是”。
2. 准备一个 ZIP 文件，其中包含：
   - 分步构建说明（在 README 文件或审阅者备注中）
   - 执行所有必要技术步骤的构建脚本
   - 操作系统和构建环境要求
   - 构建过程中使用的任何程序的所需版本和安装说明
3. 点击“浏览...”并上传你的源代码 ZIP 文件。
4. 点击“继续”。

### 步骤 3：描述你的附加组件

填写关于你的附加组件的详细信息：

![描述附加组件页面](/images/firefox/describe.png)

1. 名称：输入你的附加组件名称
2. 附加组件 URL：这将自动生成，但你可以编辑
3. 摘要：提供一个清晰简洁的说明，描述你的附加组件的功能（最多 179 个字符）
4. 描述：更长的功能和特性说明（建议 250 个字符）
5. 选择最多 2 个 Firefox 类别和 2 个 Firefox for Android 类别
6. 支持邮箱和支持网站
7. 选择一个许可证
8. 如果你的附加组件有隐私政策，请勾选框并提供政策
9. 审阅者备注：解释你的附加组件使用了 Mellowtel，并提供任何必要的背景信息

### 步骤 4：提交版本

填写完所有必填字段后，点击页面底部的“提交版本”。

### 提交后

- 你将收到一封确认邮件。
- 你的附加组件将在 AMO 上发布，但可能需要进一步审查。
- 若要在未来更新你的附加组件，请访问其在 AMO 上的页面并上传新版本。

注意：Firefox 的审查过程可能需要几天时间完成。请及时回复审查团队的任何问题。
