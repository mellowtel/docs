如果你的扩展是使用 [WXT 框架](https://wxt.dev/)构建的，你可以轻松地将 Mellowtel 集成到其中。

你可以通过在 [Github](https://github.com/yacine-bens/mellowtel-wxt-samples) 上的指南来了解如何在你的 WXT 构建的扩展中集成 Mellowtel。
