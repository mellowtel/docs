如果你的扩展是使用 [WXT 框架](https://wxt.dev/)构建的，你可以轻松地将 Mellowtel 集成到其中。

你可以通过 [Github](https://github.com/yacine-bens/mellowtel-wxt-samples) 上的指南查看如何在你的 WXT 构建的扩展中集成 Mellowtel。

如果你有任何问题或需要进一步的帮助，请加入我们的 [Discord 服务器](https://discord.com/invite/txAZp4MSDe) 或联系此仓库的开发者：[yacine.web.ext@gmail.com](mailto:yacine.web.ext@gmail.com)。
