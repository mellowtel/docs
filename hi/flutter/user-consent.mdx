## परिचय

Mellowtel को सक्रिय करने के लिए, उपयोगकर्ताओं को अपनी स्पष्ट सहमति देनी होगी। यह पैकेज ऑप्ट इन या ऑप्ट आउट करने का एक तरीका प्रदान करता है। मुख्य विचार यह है कि चूंकि यह उनका इंटरनेट बैंडविड्थ है, उन्हें यह तय करना चाहिए कि वे इसे साझा करना चाहते हैं या नहीं।

उपयोगकर्ता अपनी सहमति एक सहमति सेटिंग्स डायलॉग के माध्यम से दे सकते हैं।

### सहमति डायलॉग कॉन्फ़िगरेशन

सहमति डायलॉग Mellowtel एकीकरण का एक महत्वपूर्ण हिस्सा है। यह सुनिश्चित करता है कि उपयोगकर्ता पूरी तरह से सूचित हैं और इंटरनेट बैंडविड्थ साझा करने से पहले स्पष्ट सहमति देते हैं।

आप `Mellowtel` का एक उदाहरण बनाते समय सहमति डायलॉग कॉन्फ़िगरेशन को परिभाषित कर सकते हैं।

```dart
import 'package:mellowtel/mellowtel.dart';

final Mellowtel mellowtel = Mellowtel(
    "your_configuration_key",
    dialogConfiguration: const ConsentDialogConfiguration(
        appName: 'Your App Name',
        incentive: 'Earn rewards by sharing your internet'
        appIcon: 'assets/logo.png', // सहमति डायलॉग में प्रदर्शित वैकल्पिक ऐप आइकन। विश्वास बनाने में मदद करता है।
        acceptButtonText: "हाँ, चलो शुरू करें!", // स्वीकार बटन के लिए वैकल्पिक पाठ ("हाँ" डिफ़ॉल्ट रूप से)।
        declineButtonText: 'अभी नहीं', // अस्वीकार बटन के लिए वैकल्पिक पाठ ("नहीं" डिफ़ॉल्ट रूप से)।
        dialogTextOverride: 'कस्टम सहमति संदेश', // वैकल्पिक कस्टम या स्थानीयकृत सहमति डायलॉग संदेश।
    ),
);
```

#### सहमति डायलॉग खोलना

यदि उपयोगकर्ता ने अभी तक अपनी प्राथमिकता नहीं दी है, तो `start()` विधि कॉल करने पर सहमति डायलॉग डिफ़ॉल्ट रूप से खुलता है।

```dart
await mellowtel.start(
    ... // अन्य कॉन्फ़िगरेशन
    onOptIn: () async {
        // यदि उपयोगकर्ता ने सहमति दी है तो प्रोत्साहनों को सक्षम करें। 
    }, 
    onOptOut: () async {
        // यदि उपयोगकर्ता ने सहमति अस्वीकार/रद्द कर दी है तो प्रोत्साहनों को अक्षम करें।
    },
    showConsentDialog: true
);
```

यदि आप यह प्रबंधित करना चाहते हैं कि डायलॉग कहाँ दिखाई दे, तो आप `showConsentDialog` को false पर सेट कर सकते हैं। हम अनुशंसा करते हैं कि आप दो अलग-अलग स्थानों पर `start` विधि कॉल करें:

1. **ऐप लॉन्च पर** `showConsentDialog: false` के साथ, यदि अनुमति पहले से दी गई है तो संचालन शुरू करने के लिए।
2. **पहले "आहा" पल के बाद:** `showConsentDialog: true` के साथ सहमति का अनुरोध करने के लिए जब उपयोगकर्ता ऐप के मूल्य का अनुभव कर चुके हों।

यह दृष्टिकोण सुनिश्चित करता है कि Mellowtel तुरंत संचालन शुरू करता है जबकि उपयोगकर्ता की सहमति को एक इष्टतम समय पर प्राप्त करता है।

### सहमति सेटिंग्स डायलॉग

Mellowtel आपके उपयोगकर्ताओं के लिए पूर्ण नियंत्रण और गोपनीयता सुनिश्चित करता है। आपके उपयोगकर्ता किसी भी समय सहमति सेटिंग्स पृष्ठ से अपनी सहमति बदल सकते हैं। आप इसे अपने ऐप के सेटिंग्स पृष्ठ के भीतर एक विकल्प के रूप में प्रदान कर सकते हैं।

#### सहमति सेटिंग्स पृष्ठ दिखाएं

सहमति सेटिंग्स डायलॉग खोलने के लिए `showConsentSettingsPage()` विधि का उपयोग करें।

```dart
await mellowtel.showConsentSettingsPage(
    context,
    onOptIn: () async {
        // यदि उपयोगकर्ता ने सहमति दी है तो प्रोत्साहनों को सक्षम करें। 
    }, 
    onOptOut: () async {
        // यदि उपयोगकर्ता ने सहमति अस्वीकार/रद्द कर दी है तो प्रोत्साहनों को अक्षम करें।
    },
);
```

### मैनुअल विधियाँ (अनुशंसित नहीं)

सहमति डायलॉग और सेटिंग्स पृष्ठ के अलावा, आप `optIn()` और `optOut()` विधियों का उपयोग करके उपयोगकर्ता की सहमति को मैन्युअल रूप से प्रबंधित कर सकते हैं।

#### ऑप्ट-इन

उपयोगकर्ता को मैन्युअल रूप से ऑप्ट इन के रूप में चिह्नित करें।
```dart
await mellowtel.optIn();
```
#### ऑप्ट-आउट

उपयोगकर्ता को मैन्युअल रूप से ऑप्ट आउट के रूप में चिह्नित करें।
```dart
await mellowtel.optOut();
```
#### सहमति स्थिति की जाँच करें

आप `checkConsent()` विधि का उपयोग करके उपयोगकर्ता की वर्तमान सहमति स्थिति की जाँच कर सकते हैं।
```dart
bool? hasOptedIn = await mellowtel.checkConsent();
```
