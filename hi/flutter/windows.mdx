## NuGet CLI सेटअप

Windows पर अपने Flutter एप्लिकेशन में Mellowtel को एकीकृत करने के लिए, आपको NuGet CLI टूल की आवश्यकता होगी। इन चरणों का पालन करें:

1. [NuGet CLI डाउनलोड पेज](https://www.nuget.org/downloads) से `nuget.exe` डाउनलोड करें
2. एक फोल्डर बनाएं (उदाहरण के लिए, `C:\Tools\NuGet`) और `nuget.exe` को वहां रखें
3. अपने Windows PATH में फोल्डर जोड़ें:
   - सिस्टम प्रॉपर्टीज़ खोलें > एडवांस्ड > एनवायरनमेंट वेरिएबल्स
   - सिस्टम वेरिएबल्स के तहत, Path को संपादित करें
   - फोल्डर पथ जोड़ें (उदाहरण के लिए, `C:\Tools\NuGet`)
4. एक नए टर्मिनल में `nuget` चलाकर इंस्टॉलेशन सत्यापित करें

विस्तृत निर्देशों के लिए, [आधिकारिक NuGet CLI दस्तावेज़](https://learn.microsoft.com/en-us/nuget/install-nuget-client-tools?tabs=windows#nugetexe-cli) पर जाएं।

## समस्याएँ?

यदि आपको कोई समस्या आती है, तो कृपया समस्या निवारण युक्तियों के लिए [NuGet CLI दस्तावेज़](https://learn.microsoft.com/en-us/nuget/install-nuget-client-tools?tabs=windows#nugetexe-cli) देखें या हमारे समर्थन टीम से [Discord](https://discord.com/invite/txAZp4MSDe) पर संपर्क करें।
