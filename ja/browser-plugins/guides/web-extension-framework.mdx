あなたの拡張機能が [WXTフレームワーク](https://wxt.dev/) を使用して構築されている場合、簡単にMellowtelを統合することができます。

WXTで構築された拡張機能にMellowtelを統合する方法は、[Github](https://github.com/yacine-bens/mellowtel-wxt-samples) のガイドを参照してください。

質問がある場合やさらなるサポートが必要な場合は、私たちの[Discordサーバー](https://discord.com/invite/txAZp4MSDe)に参加するか、このリポジトリの開発者に[yacine.web.ext@gmail.com](mailto:yacine.web.ext@gmail.com)までご連絡ください。
