[クイックスタート](/browser-plugins/quickstart) ガイドに記載されているように、Mellowtel を拡張機能にインポートした後、以下の手順に従って Microsoft Edge アドオンストアに拡張機能を提出してください。

## 拡張機能の準備

1. **開発とテスト**: Microsoft Edge でテスト済みの動作するプロトタイプを用意してください。

2. **ZIP ファイルの作成**: 拡張機能ファイルを ZIP アーカイブにパッケージ化します。含めるもの:
   - 拡張機能のマニフェスト
   - 必要なすべての画像とファイル

3. **マニフェストの確認**: マニフェスト内の以下のフィールドが正確であることを確認してください:
   - `Name`
   - `Description`

## Microsoft Edge アドオンへの提出

1. **開発者アカウントの作成**:
   - まだ持っていない場合は、[パートナーセンター](https://partner.microsoft.com/dashboard/microsoftedge/public/login?ref=addons)で開発者アカウントを作成してください。
   - 開発者アカウントの作成に関する詳細な手順は、[開発者登録ガイド](https://docs.microsoft.com/microsoft-edge/extensions-chromium/publish/create-dev-account)を参照してください。

2. **新しい提出を開始**:
   - 開発者ダッシュボードにアクセスし、「新しい拡張機能を作成」を選択します。

3. **パッケージのアップロード**:
   - パッケージページで、拡張機能の ZIP ファイルをアップロードします。
   - 検証を待ち、必要に応じて問題を解決します。

   ![パッケージアップロード画面](/images/edge/packages.png)
   
   アップロード画面では、ZIP ファイルをドラッグアンドドロップするか、選択してアップロードできます。パッケージには以下が含まれていることを確認してください:
   1. 名前、バージョン、短い説明、権限、デフォルトの言語などの詳細を含むマニフェストファイル。
   2. 拡張機能に必要な画像やその他のファイル。

4. **可用性の詳細を提供**:
   - 可視性を設定（公開または非公開）。
   - 配布する市場を選択。

   ![可用性設定](/images/edge/availability.png)
   
   可用性セクションでは:
   - 公開（誰でも拡張機能を見つけて使用できる）または非公開（ストアリスティング URL を持つ人のみが使用できる）を選択します。
   - 拡張機能を利用可能にする市場を選択します。デフォルトでは、現在および将来のすべての市場に設定されています。

5. **拡張機能のプロパティを設定**:
   - カテゴリを選択。
   - プライバシーポリシーの要件を指定。
   - ウェブサイトの URL とサポート連絡先の詳細を追加。
   - 成人向けコンテンツがあるかどうかを示す。

   ![プロパティセクション](/images/edge/properties.png)
   
   プロパティセクションでは:
   - 拡張機能を最もよく表すカテゴリを選択します。
   - 拡張機能が個人情報にアクセス、収集、または送信するかどうかを示します。
   - 必要に応じてプライバシーポリシーの URL を提供します。
   - ウェブサイトの URL とサポート連絡先の詳細を追加します。

6. **ストアリスティングの詳細を追加**:
   - 説明を提供（最大10,000文字）。
   - 拡張機能ストアのロゴをアップロード（推奨サイズは300x300ピクセル）。
   - オプションのプロモーションタイルとスクリーンショットを追加。
   - 検索用語を追加して発見性を向上させます。

7. **提出を完了**:
   - 認証のためのメモを追加。
   - 「公開」をクリックして拡張機能を提出します。

## 提出後

- Microsoft は拡張機能をレビューし、最大で7営業日かかることがあります。
- レビューが完了すると通知されます。
- 問題が見つかった場合は、それを解決して再提出してください。

## 成功する提出のためのヒント

- [Microsoft Edge アドオンの開発者ポリシー](https://docs.microsoft.com/microsoft-edge/extensions-chromium/store-policies/developer-policies)に従ってください。
- 明確で簡潔な説明と高品質の画像を提供してください。
- ユーザーデータを収集する場合は、包括的なプライバシーポリシーを確保してください。
- レビュアーからのコメントや質問には迅速に対応してください。

Mellowtel の統合は提出プロセスに影響を与えないはずですが、データ収集とユーザープライバシーに関する Microsoft のポリシーに準拠していることを確認してください。

詳細な情報については、[Microsoft Edge アドオン開発者ドキュメント](https://docs.microsoft.com/microsoft-edge/extensions-chromium/publish/publish-extension)を参照してください。
