[クイックスタート](/browser-plugin/quickstart)ガイドに従って Mellowtel を拡張機能にインポートした後、以下の手順で Firefox アドオンに拡張機能を提出してください。

### ステップ 1: アドオンをアップロード

1. [アドオンデベロッパーハブ](https://addons.mozilla.org/developers/)にアクセスします。
2. 「新しいアドオンを提出」をクリックします。
3. 次のページが表示されます：

![新しいアドオン提出ページ](/images/firefox/submit_new.png)

4. 「ファイルを選択...」をクリックし、アドオンファイル（.zip、.xpi、.crx、.jar、または .xml）を選択します。
5. 「続行」をクリックし、検証が完了するのを待ちます。

### ステップ 2: ソースコードを提出

検証後、ソースコードの提出について尋ねられます：

![ソースコード提出ページ](/images/firefox/source_code.png)

1. 「ソースコードを提出する必要がありますか？」という質問に「はい」を選択します。
2. 次の内容を含む ZIP ファイルを準備します：
   - 手順ごとのビルド手順（README ファイルまたはレビューノートに記載）
   - 必要な技術的ステップを実行するビルドスクリプト
   - オペレーティングシステムとビルド環境の要件
   - ビルドプロセスで使用するプログラムの必要バージョンとインストール手順
3. 「参照...」をクリックして、ソースコード ZIP ファイルをアップロードします。
4. 「続行」をクリックします。

### ステップ 3: アドオンの説明

アドオンに関する詳細を入力します：

![アドオン説明ページ](/images/firefox/describe.png)

1. 名前: アドオンの名前を入力
2. アドオン URL: 自動生成されますが、編集可能です
3. 概要: アドオンの機能を明確かつ簡潔に説明します（最大179文字）
4. 説明: 機能と機能性の詳細な説明（推奨250文字）
5. 最大2つの Firefox カテゴリと2つの Firefox for Android カテゴリを選択
6. サポートメールとサポートウェブサイト
7. ライセンスを選択
8. アドオンにプライバシーポリシーがある場合は、チェックボックスをオンにしてポリシーを提供
9. レビュアーへのノート: アドオンが Mellowtel を使用していることを説明し、必要なコンテキストを提供

### ステップ 4: バージョンを提出

すべての必須フィールドを入力した後、ページの下部にある「バージョンを提出」をクリックします。

### 提出後

- 確認メールを受け取ります。
- アドオンは AMO に公開されますが、さらにレビューが行われる場合があります。
- 将来アドオンを更新するには、AMO のページにアクセスして新しいバージョンをアップロードしてください。

注意: Firefox のレビュー プロセスには数日かかる場合があります。レビューチームからの質問には迅速に対応してください。
