**MellowtelはAIの世界で開発者とクリエイターのためのオープンソースの収益化プラットフォームです。** SDKをブラウザ拡張機能、デスクトップアプリ、ウェブサイト、またはモバイルアプリに組み込むと、ユーザーは信頼できるパートナーと未使用のインターネット帯域幅の一部を共有することに同意できます。あなたは収益の一部を得ることができます。

## 統合の様子

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

これがブラウザ拡張機能のための統合全体です。他のプラットフォームも同様です。以下のクイックスタートを参照してください。

## 収益の仕組み

分配は**開発者に55%、Mellowtelに45%**です。スケールが増すにつれて、より多くのサービスを提供し、開発者の取り分を増やすことができるようになります。将来的には、この収益の一部をユーザーと直接共有できる機能が追加されるかもしれません。それまでの間、オプトインしたユーザーに独占的な機能やコンテンツを提供して報酬を与えることができます。
