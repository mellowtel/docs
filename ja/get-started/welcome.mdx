**Mellowtelは、AIの世界で開発者とクリエイターのためのオープンソースの収益化プラットフォームです。** SDKをブラウザ拡張機能、デスクトップアプリ、ウェブサイト、またはモバイルアプリに組み込むと、ユーザーは信頼できるパートナーと未使用のインターネット帯域幅の一部を共有することを選択できます。あなたは収益の一部を得ることができます。

## 統合のイメージ

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

これがブラウザ拡張機能のための全統合です。他のプラットフォームも同様です。以下のクイックスタートを参照してください。

## 収益の仕組み

分配は**開発者に55%、Mellowtelに45%**です。規模が拡大するにつれて、より多くのサービスを提供し、開発者のシェアを増やすことができるようになります。将来的には、この収益の一部をユーザーと直接共有できる機能があるかもしれません。その間、オプトインしたユーザーには、独占的な機能やコンテンツで報酬を与えることができます。
