Если ваше расширение создано с использованием [фреймворка WXT](https://wxt.dev/), вы можете легко интегрировать Mellowtel в него.

Вы можете узнать, как интегрировать Mellowtel в расширение, созданное с помощью WXT, следуя руководству на [Github](https://github.com/yacine-bens/mellowtel-wxt-samples).
