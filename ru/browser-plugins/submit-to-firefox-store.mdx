После импорта Mellowtel в ваше расширение, как описано в руководстве [Quick Start](/browser-plugin/quickstart), выполните следующие шаги для отправки вашего расширения в Firefox Add-ons.

### Шаг 1: Загрузите ваше дополнение

1. Перейдите на [Add-ons Developer Hub](https://addons.mozilla.org/developers/).
2. Нажмите "Submit a New Add-on".
3. Вы увидите эту страницу:

![Страница отправки нового дополнения](/images/firefox/submit_new.png)

4. Нажмите "Select a file..." и выберите файл вашего дополнения (.zip, .xpi, .crx, .jar или .xml).
5. Нажмите "Continue" и дождитесь завершения проверки.

### Шаг 2: Отправка исходного кода

После проверки вас спросят об отправке исходного кода:

![Страница отправки исходного кода](/images/firefox/source_code.png)

1. Выберите "Yes" на вопрос "Do you need to submit source code?".
2. Подготовьте ZIP файл, содержащий:
   - Пошаговые инструкции по сборке (в файле README или примечаниях для рецензента)
   - Скрипт сборки, выполняющий все необходимые технические шаги
   - Требования к операционной системе и среде сборки
   - Требуемую версию и инструкции по установке для любых программ, используемых в процессе сборки
3. Нажмите "Browse..." и загрузите ваш ZIP файл с исходным кодом.
4. Нажмите "Continue".

### Шаг 3: Описание вашего дополнения

Заполните информацию о вашем дополнении:

![Страница описания дополнения](/images/firefox/describe.png)

1. Name: Введите название вашего дополнения
2. Add-on URL: Этот адрес будет сгенерирован автоматически, но вы можете его изменить
3. Summary: Предоставьте четкое и краткое объяснение того, что делает ваше дополнение (максимум 179 символов)
4. Description: Более подробное объяснение функций и возможностей (рекомендуется 250 символов)
5. Выберите до 2 категорий Firefox и 2 категорий Firefox для Android
6. Email поддержки и сайт поддержки
7. Выберите лицензию
8. Если у вашего дополнения есть политика конфиденциальности, отметьте галочку и предоставьте политику
9. Примечания для рецензентов: Объясните, что ваше дополнение использует Mellowtel и предоставьте необходимый контекст

### Шаг 4: Отправка версии

После заполнения всех обязательных полей нажмите "Submit Version" внизу страницы.

### После отправки

- Вы получите подтверждение по электронной почте.
- Ваше дополнение будет опубликовано на AMO, но может подлежать дальнейшему рассмотрению.
- Чтобы обновить ваше дополнение в будущем, посетите его страницу на AMO и загрузите новую версию там.

Примечание: Процесс проверки Firefox может занять несколько дней. Быстро отвечайте на любые вопросы от команды рецензентов.
