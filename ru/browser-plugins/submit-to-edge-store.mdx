После импорта Mellowtel в ваше расширение, как описано в руководстве [Быстрый старт](/browser-plugins/quickstart), следуйте этим шагам, чтобы отправить ваше расширение в магазин Microsoft Edge Add-ons.

## Подготовьте ваше расширение

1. **Разработка и тестирование**: Убедитесь, что у вас есть рабочий прототип вашего расширения, протестированный в Microsoft Edge.

2. **Создайте ZIP-файл**: Упакуйте файлы вашего расширения в архив ZIP, включая:
   - Манифест расширения
   - Все необходимые изображения и файлы

3. **Проверьте манифест**: Убедитесь, что следующие поля в вашем манифесте точны:
   - `Name`
   - `Description`

## Отправка в Microsoft Edge Add-ons

1. **Создайте учетную запись разработчика**:
   - Если у вас её нет, создайте учетную запись разработчика на [Partner Center](https://partner.microsoft.com/dashboard/microsoftedge/public/login?ref=addons).
   - Для подробных инструкций по созданию учетной записи разработчика, смотрите [Руководство по регистрации разработчика](https://docs.microsoft.com/microsoft-edge/extensions-chromium/publish/create-dev-account).

2. **Начните новое отправление**:
   - Перейдите на панель разработчика и выберите "Создать новое расширение".

3. **Загрузите ваш пакет**:
   - На странице Пакеты загрузите ваш ZIP-файл расширения.
   - Подождите проверки и устраните любые проблемы, если необходимо.

   ![Экран загрузки пакета](/images/edge/packages.png)
   
   Экран загрузки позволяет перетащить ваш ZIP-файл или выбрать его. Убедитесь, что ваш пакет содержит:
   1. Файл манифеста с такими деталями, как имя, версия, краткое описание, разрешения и язык по умолчанию.
   2. Изображения и другие файлы, которые требуются вашему расширению.

4. **Укажите детали доступности**:
   - Установите видимость (Публичная или Скрытая).
   - Выберите рынки для распространения.

   ![Настройки доступности](/images/edge/availability.png)
   
   В разделе Доступность:
   - Выберите между Публичной (любой может найти и использовать расширение) или Скрытой (только те, у кого есть URL листинга в магазине, могут его использовать).
   - Выберите рынки, где ваше расширение будет доступно. По умолчанию установлено для всех текущих и будущих рынков.

5. **Установите свойства расширения**:
   - Выберите категорию.
   - Укажите требования к политике конфиденциальности.
   - Добавьте URL сайта и контактные данные поддержки.
   - Укажите, если есть контент для взрослых.

   ![Раздел свойств](/images/edge/properties.png)
   
   В разделе Свойства:
   - Выберите категорию, которая лучше всего описывает ваше расширение.
   - Укажите, если ваше расширение получает доступ, собирает или передает личную информацию.
   - Предоставьте URL политики конфиденциальности, если требуется.
   - Добавьте URL вашего сайта и контактные данные поддержки.

6. **Добавьте детали листинга в магазине**:
   - Предоставьте описание (до 10,000 символов).
   - Загрузите логотип для Магазина расширений (рекомендуется 300x300 пикселей).
   - Добавьте дополнительные промо-плитки и скриншоты.
   - Включите поисковые термины для улучшения обнаруживаемости.

7. **Завершите отправку**:
   - Добавьте любые заметки для сертификации.
   - Нажмите "Опубликовать", чтобы отправить ваше расширение.

## После отправки

- Microsoft проверит ваше расширение, что может занять до семи рабочих дней.
- Вы получите уведомление, когда проверка будет завершена.
- Если будут найдены проблемы, устраните их и отправьте снова.

## Советы для успешной отправки

- Следуйте [Политикам разработчиков для Microsoft Edge Add-ons](https://docs.microsoft.com/microsoft-edge/extensions-chromium/store-policies/developer-policies).
- Предоставляйте четкие, лаконичные описания и качественные изображения.
- Убедитесь, что ваша политика конфиденциальности является исчерпывающей, если вы собираете какие-либо данные пользователей.
- Быстро отвечайте на любые комментарии или вопросы рецензентов.

Помните, интеграция Mellowtel не должна влиять на процесс отправки, но убедитесь, что она соответствует политикам Microsoft в отношении сбора данных и конфиденциальности пользователей.

Для более подробной информации посетите [Документацию разработчика Microsoft Edge Add-ons](https://docs.microsoft.com/microsoft-edge/extensions-chromium/publish/publish-extension).
