Mellowtel предоставляет готовую страницу отзывов, которую вы можете использовать для сбора отзывов от ваших пользователей, чтобы быстрее вносить изменения и улучшать ваш продукт.
На данный момент страница доступна только для браузерных плагинов. Страница отзывов похожа на [эту](https://www.mellowtel.com/uninstall-feedback/?extension_id=testId&configuration_key=testConfiguration), передавая динамически иконку и название вашего плагина.

Затем вы можете просматривать отзывы на панели управления Mellowtel и использовать их для улучшения вашего браузерного плагина.

ВАЖНО: Вы сможете просматривать отзывы на панели управления Mellowtel только если у вас есть плагин, опубликованный в веб-магазине, а не в режиме разработки.
Если вы все еще находитесь в стадии разработки и хотите протестировать страницу отзывов, пожалуйста, свяжитесь с нами через чат.

Вот как вы можете добавить страницу отзывов в ваш браузерный плагин:


## Добавьте страницу отзывов в ваш плагин

Страница отзывов размещена на Mellowtel и доступна через URL, который вы можете сгенерировать с помощью метода `generateFeedbackLink()` из пакета Mellowtel JS.

Если вы хотите напрямую открыть страницу отзывов в новой вкладке, вы можете использовать метод `generateAndOpenFeedbackLink()`, и он откроет страницу, похожую на [эту](https://www.mellowtel.com/uninstall-feedback/?extension_id=testId&configuration_key=testConfiguration), динамически передавая иконку и название вашего плагина.

Вы можете открыть страницу отзывов, вызвав метод `generateFeedbackLink()` из фонового скрипта или контентного скрипта вашего расширения, когда пользователь удаляет ваш браузерный плагин.

Вот как вы можете это сделать.

Откройте ваш файл `service worker` (также известный как фоновый скрипт) и добавьте следующий код:

Не забудьте заменить `<YOUR_CONFIGURATION_KEY>` на ваш фактический ключ конфигурации Mellowtel. Если у вас еще нет ключа конфигурации, вы можете получить его, зарегистрировавшись на [Mellowtel](https://www.mellowtel.com/mellowtel-auth/).

```javascript
import Mellowtel from "mellowtel";

let mellowtel;

(async () => {
    mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
    await mellowtel.initBackground();
})();

// Открыть страницу отзывов, когда пользователь удаляет расширение
chrome.runtime.onInstalled.addListener(async function(details) {
    console.log("Расширение установлено или обновлено");
    // Если вы хотите обрабатывать установку и обновления по-разному
    /**
    if(details.reason === "install"){
        // вызовите функцию для обработки первой установки
    } else if(details.reason === "update") {
        // вызовите функцию для обработки обновления
    }
    **/
    // await mellowtel.generateAndOpenOptInLink(); для управления настройками пользователей. Для получения дополнительной информации: документация по opt-in-out
    const uninstallURl = await mellowtel.generateFeedbackLink();
    chrome.runtime.setUninstallURL(uninstallURl);
});
```
