## Введение

Для активации Mellowtel пользователи должны дать свое явное согласие. Пакет предоставляет возможность включения или отключения. Основная идея заключается в том, что поскольку это их интернет-канал, они должны решать, хотят ли они его делиться или нет.

Пользователи могут дать свое согласие через диалог настроек согласия.

### Конфигурация диалога согласия

Диалог согласия является важной частью интеграции Mellowtel. Он гарантирует, что пользователи полностью информированы и дают явное согласие перед тем, как делиться своим интернет-каналом.

Вы можете определить конфигурацию диалога согласия при создании экземпляра `Mellowtel`

```dart
import 'package:mellowtel/mellowtel.dart';

final Mellowtel mellowtel = Mellowtel(
    "your_configuration_key",
    dialogConfiguration: const ConsentDialogConfiguration(
        appName: 'Название вашего приложения',
        incentive: 'Зарабатывайте награды, делясь своим интернетом'
        appIcon: 'assets/logo.png', // Необязательная иконка приложения, отображаемая в диалоге согласия. Помогает завоевать доверие.
        acceptButtonText: "Да, поехали!", // Необязательный текст для кнопки принятия ("Да" по умолчанию).
        declineButtonText: 'Не сейчас', // Необязательный текст для кнопки отказа ("Нет" по умолчанию).
        dialogTextOverride: 'Пользовательское сообщение о согласии', // Необязательное пользовательское или локализованное сообщение диалога согласия.
    ),
);
```

#### Открытие диалога согласия

Диалог согласия по умолчанию открывается при вызове метода `start()`, если пользователь еще не указал свои предпочтения.

```dart
await mellowtel.start(
    ... // другая конфигурация
    onOptIn: () async {
        // Включите стимулы, если пользователь дал согласие. 
    }, 
    onOptOut: () async {
        // Отключите стимулы, если пользователь отказался/отозвал согласие.
    },
    showConsentDialog: true
);
```

Вы можете установить `showConsentDialog` в значение false, если хотите управлять, где будет отображаться диалог. Мы рекомендуем вызывать метод `start` в двух разных местах:

1. **При запуске приложения** с `showConsentDialog: false`, чтобы начать работу, если разрешение уже предоставлено.
2. **После первого "Ага" момента:** с `showConsentDialog: true`, чтобы запросить согласие после того, как пользователи оценят ценность приложения.

Этот подход гарантирует, что Mellowtel начнет работать немедленно, получая согласие пользователя в оптимальное время.

### Диалог настроек согласия

Mellowtel обеспечивает полный контроль и конфиденциальность для ваших пользователей. Ваши пользователи могут изменить свое согласие в любое время на странице настроек согласия. Вы можете предоставить его как опцию на странице настроек вашего приложения.

#### Показать страницу настроек согласия

Используйте метод `showConsentSettingsPage()`, чтобы открыть диалог настроек согласия.

```dart
await mellowtel.showConsentSettingsPage(
    context,
    onOptIn: () async {
        // Включите стимулы, если пользователь дал согласие. 
    }, 
    onOptOut: () async {
        // Отключите стимулы, если пользователь отказался/отозвал согласие.
    },
);
```

### Ручные методы (не рекомендуется)

В дополнение к диалогу согласия и странице настроек, вы можете вручную управлять согласием пользователя, используя методы `optIn()` и `optOut()`.

#### Подключение

Вручную отметьте пользователя как подключенного.
```dart
await mellowtel.optIn();
```
#### Отключение

Вручную отметьте пользователя как отключенного.
```dart
await mellowtel.optOut();
```
#### Проверка статуса согласия

Вы можете проверить текущий статус согласия пользователя, используя метод `checkConsent()`.
```dart
bool? hasOptedIn = await mellowtel.checkConsent();
```
