## Настройка NuGet CLI

Чтобы интегрировать Mellowtel в ваши Flutter приложения на Windows, вам понадобится инструмент NuGet CLI. Следуйте этим шагам:

1. Скачайте `nuget.exe` с [страницы загрузки NuGet CLI](https://www.nuget.org/downloads)
2. Создайте папку (например, `C:\Tools\NuGet`) и поместите туда `nuget.exe`
3. Добавьте папку в PATH Windows:
   - Откройте Свойства системы > Дополнительно > Переменные среды
   - В разделе Системные переменные, отредактируйте Path
   - Добавьте путь к папке (например, `C:\Tools\NuGet`)
4. Проверьте установку, запустив `nuget` в новом терминале

Для подробных инструкций посетите [официальную документацию NuGet CLI](https://learn.microsoft.com/en-us/nuget/install-nuget-client-tools?tabs=windows#nugetexe-cli).
