**Mellowtel — это платформа монетизации с открытым исходным кодом для разработчиков и создателей в мире ИИ.** Добавьте SDK в ваше расширение для браузера, настольное приложение, веб-сайт или мобильное приложение, и ваши пользователи смогут делиться частью своего неиспользуемого интернет-трафика с доверенными партнерами. Вы получаете долю от дохода.

## Как выглядит интеграция

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

Это вся интеграция для расширения браузера. Другие платформы аналогичны. См. быстрые старты ниже.

## Как работает доход

Доля составляет **55% для разработчиков, 45% для Mellowtel**. По мере роста мы сможем предлагать больше услуг и увеличивать долю разработчиков. В дальнейшем могут появиться функции, позволяющие делиться частью этого дохода непосредственно с вашими пользователями; пока вы можете вознаграждать подключенных пользователей эксклюзивными функциями или контентом.
