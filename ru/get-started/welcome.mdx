**Mellowtel — это платформа с открытым исходным кодом для монетизации, предназначенная для разработчиков и создателей в мире ИИ.** Вставьте SDK в свое браузерное расширение, настольное приложение, сайт или мобильное приложение, и ваши пользователи смогут согласиться делиться частью своего неиспользуемого интернет-трафика с доверенными партнерами. Вы получаете долю дохода.

## Как выглядит интеграция

```bash
npm install mellowtel
```

```javascript
import Mellowtel from "mellowtel";

const mellowtel = new Mellowtel("<YOUR_CONFIGURATION_KEY>");
await mellowtel.initBackground();
await mellowtel.generateAndOpenOptInLink();
```

Это вся интеграция для браузерного расширения. Другие платформы аналогичны. См. быстрые старты ниже.

## Как работает доход

Доля составляет **55% для разработчиков, 45% для Mellowtel**. По мере нашего роста мы сможем предлагать больше услуг и увеличивать долю разработчиков. В дальнейшем могут появиться функции, позволяющие делиться частью этого дохода напрямую с вашими пользователями; пока вы можете вознаграждать пользователей, согласившихся участвовать, эксклюзивными функциями или контентом.
