Mellowtel основан на четырех принципах. Прочитайте их один раз, и вы поймете каждое поведение, настройку по умолчанию и выбор API в остальной части документации.

<CardGroup cols={2}>
  <Card title="Подписка / Отписка" icon="signature" href="/concepts/opt-in-out">
    Пользователи должны дать явное согласие перед тем, как будет передан любой трафик, и могут отозвать его в любое время.
  </Card>
  <Card title="Конфиденциальность" icon="lock" href="/concepts/privacy">
    Запросы выполняются в изолированной среде без учетных данных. Ни куки, ни хранилище, ни личные данные никогда не покидают устройство.
  </Card>
  <Card title="Ограничение скорости" icon="stopwatch" href="/concepts/rate-limiting">
    SDK автоматически регулирует себя, чтобы пользовательский опыт просмотра никогда не страдал.
  </Card>
  <Card title="Открытый исходный код" icon="github" href="https://github.com/mellowtel-inc/">
    Каждая строка кода на GitHub. Если документация неясна, исходный код — это истина.
  </Card>
</CardGroup>
