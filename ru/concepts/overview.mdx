Mellowtel основан на четырех принципах. Прочитав их один раз, ты поймешь каждое поведение, значение по умолчанию и выбор API в остальной части документации.

<CardGroup cols={2}>
  <Card title="Подключение / Отключение" icon="signature" href="/concepts/opt-in-out">
    Пользователи должны дать явное согласие перед тем, как будет поделена любая полоса пропускания, и могут отозвать его в любое время.
  </Card>
  <Card title="Конфиденциальность" icon="lock" href="/concepts/privacy">
    Запросы выполняются в изолированной среде без учетных данных. Никакие cookies, хранилища или личные данные никогда не покидают устройство.
  </Card>
  <Card title="Ограничение скорости" icon="stopwatch" href="/concepts/rate-limiting">
    SDK автоматически регулирует себя, чтобы пользовательский опыт просмотра никогда не был нарушен.
  </Card>
  <Card title="Открытый исходный код" icon="github" href="https://github.com/mellowtel-inc/">
    Каждая строка находится на GitHub. Если документация неясна, исходный код является основой.
  </Card>
</CardGroup>
