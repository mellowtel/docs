Мы заменяем устаревший `configuration_key` новым идентификатором, называемым **Integration ID**. Если вы уже используете Mellowtel, вам нужно будет обновить строку, которую вы передаете в SDK.

## Кратко

- Каждая интеграция теперь имеет свой собственный **Integration ID**, с префиксом `intgr-`.
- Старый `configuration_key` все еще работает, но будет устаревшим.
- Единственное изменение в коде — это строка, передаваемая в `new Mellowtel(...)`.

## Ключ конфигурации vs Integration ID

| | Ключ конфигурации (устаревший) | Integration ID (новый) |
|---|---|---|
| Область действия | Один ключ на аккаунт | Один ID на интеграцию |
| Формат | Непрозрачная строка | `intgr-XXXXXXXXXX` |
| Статистика | Объединена для всех приложений | Для каждого приложения |

## Почему мы вносим это изменение

Mellowtel теперь используется в браузерных расширениях, настольных приложениях, мобильных приложениях и на веб-сайтах. С одним общим ключом команды не могли определить, какая платформа действительно приносит доход.

Один Integration ID на приложение решает эту проблему. После миграции вы получаете доступ к статистике по каждому приложению, доходам по каждому приложению и функциям панели управления, которые мы разрабатываем, чтобы помочь вам развивать каждое приложение независимо.

## Что вам нужно изменить

Только строку, которую вы передаете в `new Mellowtel(...)`.

```javascript
// До
const mellowtel = new Mellowtel("<your_configuration_key>");

// После
const mellowtel = new Mellowtel("intgr-MG8tHhnkDN");
```

Все остальное остается без изменений.

## Получение вашего Integration ID

1. Откройте [страницу интеграций](https://www.mellowtel.com/dashboard/integrations) в вашей панели управления.
2. Создайте одну интеграцию для каждого приложения, выбрав соответствующий тип (расширение, настольное, мобильное, веб-сайт).
3. Скопируйте Integration ID и вставьте его в инициализацию вашего SDK.

## Обратная совместимость

Существующий код продолжает работать. Мы объявим дату устаревания с достаточным уведомлением до удаления устаревшего ключа. Миграция раньше открывает функции для каждого приложения, так что нет причин ждать.
